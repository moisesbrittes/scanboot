📤 Upload de Imagem

Para testar upload, use a API diretamente:

curl -X POST http://localhost:3002/api/documents/upload-process \
  -F "file=@imagem.jpg" \
  -F "dpi=300"

ou acesse a interface web completa em React

🔗 Endpoints Disponíveis:

Método Endpoint Descrição
GET /api/documents Listar documentos
POST /api/documents/upload-process Upload + OCR
GET /api/scan/status Status do scanner
POST /api/documents/:id/share-email Compartilhar por email

📱 Interface React Completa

O App.jsx contém uma interface React completa com:

  • Upload com OCR automático
  • Lista de documentos com busca
  • Visualizador de documentos
  • Compartilhamento (Email/WhatsApp)
  • WebSocket em tempo real

ℹ️ Informações do Sistema

📊 Backend

Express.js + Supabase PostgreSQL

Porta: 3002

WebSocket: ws://localhost:3002

📦 Banco de Dados

Supabase PostgreSQL

Tables: documents, sharing_history

Storage: scans/

🎨 Frontend

React 18 + CSS3

Arquivo: App.jsx

Estilos: App.css

🚀 Para usar a interface web completa:

  1. Abra um novo terminal
  2. Clone este projeto em uma pasta web (ex: criar um projeto Vite)
  3. Ou acesse diretamente via navegador usando a API HTTP
